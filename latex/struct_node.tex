\doxysubsection{Detailed Description}
Represents a node in the CSMA simulator. 

This struct stores information about a node in the CSMA (Carrier Sense Multiple Access) simulator. It includes the node\textquotesingle{}s ID, backoff value, time count, backoff count, and collision count.
